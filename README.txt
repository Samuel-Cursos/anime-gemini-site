# Anime Gemini Temas

Projeto separado em HTML, CSS e JavaScript.

## Arquivos

- `index.html` = estrutura do site
- `style.css` = visual e temas
- `script.js` = Gemini API, chat e troca de tema

## Como usar

1. Abra a pasta no VS Code.
2. Abra o `index.html` no navegador ou com Live Server.
3. Cole sua API Key do Gemini no campo da tela.
4. Escolha o modelo.
5. Converse.

## Temas

- Solo Leveling
- Naruto
- Pokémon

## Aviso

Para aula/teste, usar API Key no navegador funciona.
Para site publicado, use backend para esconder a chave.
